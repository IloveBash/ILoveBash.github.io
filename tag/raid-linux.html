RAID Linux

Pré-requis

# Installation debian paquet mdadm
$ aptitude install mdadm -y

# Installation distribution RHEL mdadm
$ yum install mdadm

Mise en place du RAID

Première étape, configurer le RAID par l'intermédiaire de l'outil mdadm, dans notre exemple il s'agit d'un RAID de niveau 1 basé sur trois disques dont un réservé comme spare.

$ mdadm --create --verbose /dev/md0 --level=1 --raid-devices=2 --spare-devices=1 /dev/sd[c-e]

Suivre l'opération en temps réel pour contrôler le bon déroulement de la création

$ watch mdadm --detail --verbose /dev/md0

Ensuite, il nous faut renseigner le fichier mdadm.conf, attention l'emplacement peut différer selon les distributions, dans notre exemple nous réalisons l'opération sous une distribution Debian

$ mdadm --detail --brief /dev/md0 >> /etc/mdadm/mdadm.conf

Contrôler les paramétrages de la mise en place de notre RAID

$ cat /proc/mdstat
$ mdadm --verbose --scan

Mettre un filesystem sur notre RAID

$ mkfs.ext4 /dev/md0

Renseigner le fstab pour que le montage soit effectif à chaque démarrage

$ blkid -o export /dev/md0 | grep UUID >> /etc/fstab

Vérifier que le montage soit opérationnel avec la commande mount

$ mount /dev/md0

Quelques précisions

Simuler une panne dans la solution RAID

$ mdadm /dev/md0 --fail /dev/sde

Arrêter la solution RAID

$ mdadm --stop /dev/md0

Ajouter un spare (ou autre) à notre solution RAID

$ mdadm /dev/md0 --add /dev/sdf

Supprimer un disque de notre solution RAID

$ mdadm /dev/md0 --remove /dev/sdf

Remettre à zéro les superblocs

$ mdadm --zero-superbloc /dev/sd[c-f]
